# Zanok - Mobile Game
Word addition and guessing game.

## Installation

```bash
npm install
# or
yarn install
# or
bun install
```

## Usage

```bash
npm start
# or
yarn start
# or
bun run start
``` 

## Building

```bash
eas build --platform ios
# or
eas build --platform android
```
